Sobre Mí

Welcome!

¡Hola! ¿como estas? mi nombre es Sergio.

Soy un apasionado autodidacta y cursante de la carrera de Ciencias de la Computacion en la Universidad Nacional de Cordoba.

He realizado algunos proyectos web, usando en el FrontEnd: HTML5, CSS3, Grid, Flexbox, Bootstrap, TypeScript, JS, Angular y Bootstrap, todos responsive design, en la parte de base de datos SQL Server y MySQL conectandolo al BackEnd a traves de MVC, consumiendo alguna ApiRest que haya creado, escrita en C# mediante ASP.Net, o Python mediante Django, y comunicarlo con peticiones HTTP en JSON, tambien uso CORS para conectar el Frontend y Backend.

Como QA Tester diseño y efectuo Test Cases exploratorios, de regresion, smoke tests, y automatizo pruebas con Selenium, Behave y Python, para escenarios reproducibles, aumentando el coverage del testing en la aplicacion (Web, Mobile o Desktop).

Mis Habilidades

Algunas de las tecnologías que he usado y aprendo activamente

QA Testing

Diseño y ejecucion de tests cases, test exploratorios, de regresiones, smoke tests, reporte de bugs, test manuales y automaticos con Selenium, Behave, Gherkin y Python.

Angular

Angular es un framework de Javascript enfocado al fronted de una aplicacion, tiene estructura SPA por lo que es muy rapido, pero a su vez muy completo y posee una amplia comunidad.

Bootstrap

Es un framework que contiene plantillas de diseño, formularios, botones, cuadros, y otros elementos de diseño basado en HTML y CSS para desarrollo Front-End.

SQL Server

Aca creo, gestiono y administro las bases de datos, asi como tambien sus stored procedures para facilitar las consultas enviadas desde el Fronted hacia el Backend.

JIRA

Es un potente software de gestión de proyecto, puedo ver tableros Kanban desde cualquier sitio, activar notificaciones, visualizar metricas del equipo y con Zephyr automatizar ciclos de prueba.

GIT-- GITHUB-- GITLAB

Git es un software de control de versiones, para trabajar sobre un codigo estable entre varios desarrolladores, sumando codigo desde cualquier parte del mundo.

ReactJs

React es una libreria de JavaScript, estoy practicando con esta herramienta, que esta entre las mas solicitadas del mercado.

Android Studio

Pruebas en aplicaciones nativas de android, debuguear errores y buildear versiones.

JavaScript

Es el lenguaje orientado a objetos mas usado, permite realizar efectos y acciones dinámicas, pronto practicare Node Js.

HTML5

HTML es un lenguaje de marcas de hipertexto que mediante etiquetas ordenadas forma la estructura de una página web.

CSS3

CSS (hojas de estilo en cascada) estiliza el codigo HTML, referencia, da efectos y diagrama sus elementos (Grid o Flexbox).

Responsive Design

Responsive Design permitiéndo hacer que nuestros sitios web puedan adaptarse a cualquier dispositivo móvil o desktop.

Python

Es un lenguaje muy popular, util y potente, suelo usarlo para escribir codigo para pruebas de automatizaciones, con Selenium, Behave y Cucumber (escrito en Gherkin).

Wordpress

Es un Sistema de gestión de contenidos, desarrollado en PHP, es una de las principales herramientas para la creación de páginas web comerciales.
